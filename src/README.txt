All the files are according to how it was asked for in the assignment.

To take care of the external libraries used, there is a makefile implemented.

Kindly use full/absolute path where path are expected.

Both forms of query files are compatible.

use 'python3 printdict.py <dictfilepath>' to print the whole dictionary. To print the dictionary partially you can import the funtion inside printdict.py and pass argumnets ranging from lexographical limits to limits on position in dictionary. Some corner error cases are not handled so do not put the maximum value less than minimum or similar things.

python version around 3.8 will work well.
